# Fab Ebook

## Live URL

https://fab-e-book-platform.vercel.app

## Project Purpose

Fab Ebook is an online ebook platform where users can browse, purchase, and manage digital books. The platform provides a smooth reading and purchasing experience with secure authentication and user-friendly design.

## Key Features

* User Authentication (Google Sign In & Email Login)
* Browse and search ebooks
* View detailed book information
* Add books to cart
* Secure checkout process
* User dashboard
* Responsive design for mobile and desktop
* Book management system for admins
